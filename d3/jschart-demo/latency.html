Data shown below is for a 2 hour baseline test of a 128 VM CEPH cluster. Data sets labeled with '-histogram' use the *_clat_hist* files generated by fio whereas '-clats' data sets use the standard *_clat.* files containing every single latency sample. Histogram statistics are computed over 60 second intervals, resulting in the more smooth lines seen. clat statistics are computed over 10 second intervals to emphasize the fact that the histogram statistics are accurately tracking the "true" statistics traditionally taken from fio log files.
